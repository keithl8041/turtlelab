Turtle Flow AI turns plain-English drawing ideas into safe turtle code, animates the result on the canvas, and explains what the program is doing so you can inspect it, learn from it, and improve it.

Start with a prompt, let the app suggest a drawing program, then review the generated code, edit it by hand, replay the turtle path, and save versions you want to keep. If you want live AI generation through your own provider account, you can add a session-only API key here before entering the studio.

AI setup

Bring your own API key if you want the app to use your provider account. Keys stay only in server memory for this session. Only limited functionality is available without an API key.

      🐢 Commands
      
          
        
      
        
      Type these in the Code window, one per line.
      
        
      
          
      
            
      Moving
      
            
      
              
      forward(n)
      Move forward n steps (1–500)
      
              
      backward(n)
      Move backward n steps (1–500)
      
              
      goto(x, y)
      Jump to position x, y on the canvas
      
              
      home()
      Go back to the centre of the canvas
      
            
      
          
      
          
      
            
      Turning
      
            
      
              
      left(deg)
      Turn left by deg degrees
      
              
      right(deg)
      Turn right by deg degrees
      
              
      setheading(deg)
      Point in a fixed direction (0 = right, 90 = up)
      
            
      
          
      
          
      
            
      Drawing
      
            
      
              
      penup()
      Lift the pen — moving won't draw
      
              
      pendown()
      Put the pen down — moving draws a line
      
              
      pensize(n)
      Set line thickness to n pixels
      
              
      circle(r)
      Draw a circle with radius r (max 300)
      
              
      dot(d)
      Draw a filled dot of diameter d
      
              
      clear()
      Wipe the canvas clean
      
            
      
          
      
          
      
            
      Colour & Fill
      
            
      
              
      color("name")
      Set colour — use a name or #rrggbb
      
              
      beginfill()
      Start filling the next shape
      
              
      endfill()
      Finish filling the shape
      
            
      
            
      Colour names: black, white, red, green, blue, yellow, orange, purple, pink, brown, gray
      
          
      
          
      
            
      Repeat & Comments
      
            
      
              
      
                
      repeat(n) { … }
      
                
      Run the commands inside n times (1–50)
      
              
      
              
      # your note
      A comment — ignored by the turtle
      
            
      
            
      repeat(4) {
  forward(100)
  right(90)
}
